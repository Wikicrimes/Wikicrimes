1) O que é WikiCrimes Mobile?

WikiCrimes Mobile é uma aplicação para celulares que rodem aplicações JAVA (MIDP 2.0/CLDC 1.1). A aplicação oferece um serviço que informa ao usuário os crimes de uma localidade permitindo-o avaliar se a área é ou não segura (vide imagem a seguir). O serviço é prestado a partir da base de dados do sistema de registro colaborativo de crimes WikiCrimes.org (www.wikicrimes.org). É importante ressaltar que para cada consulta é efetuada uma conexão com a internet e o envio e recebimento de dados do servidor de WikiCrimes. Dependendo do plano/contrato que você tenha com a sua operadora, este acesso e transmissão de dados pode incidir em custos. O custo médio para utilizar o aplicativo fazendo uma consulta a base de dados do WikiCrimes.org através de conexão Wap GPRS varia entre R$ 0,02 (dois centavos de Real) e R$ 0,05 (cinco centavos de Real). Estes valores foram calculados para um celular pré-pago da operadora Oi-Ceará. Se o seu celular acessar pontos de acessos WiFi de rede sem fio o custo das consultas será zero. Sugerimos uma consulta a base de dado do WikiCrimes.org (www.wikicrimes.org) para verificar se a área que você deseja pesquisar possui um bom índice de colaboração e portanto uma boa quantidade de crimes cadastrados.

 

 

 

2) O que é necessário para ter WikiCrimes Mobile?

Para baixar e instalar WikiCrimes Mobile você deverá preencher um formulário e poderá utilizar quantas vezes e por quanto tempo quiser. Entretanto a aplicação só poderá ser instalada uma única vez já que utiliza uma chave de acesso. Após o preenchimento do formulário seu navegador será redirecionado para a página de download onde você poderá baixar o WikiCrimes Mobile para seu computador e posteriormente copiar o aplicativo para o seu celular. Lembre-se que dependendo do tipo de conexão que seu celular possui com a internet você irá pagar pelo download do aplicativo.

 

 

 

3) Como instalar a aplicação em meu celular?

Uma vez tendo pago R$ 5,00 seu navegador será redirecionado para a página de download do WikiCrimes Mobile. Uma vez nessa página você poderá baixar o aplicativo para o seu computador e posteriormente copia-lo para o seu celular, ou baixar ele diretamente pelo celular. Na maioria dos celulares a instalação é automatica uma vez que o seu celular detecta que vc está baixando ou copiando um aplicativo JAVA, podendo a aplicação ser instalada na pasta de Jogos do seu celular ou em minhas aplicações dependendo do celular. Outros aparelhos requerem que seja utilizado um software que foi distribuído pelo fabricante para instalação de aplicativos JAVA. Neste caso deve-se baixar o aplicativo WikiCrimes Mobile para o seu desktop e apartir do software distribuído pelo fabricante realizar a inslação do mesmo em seu celular. Note que é um requisito para a instalação do WikiCrimes Mobile que seu celular execute aplicativos JAVA (J2ME MIDP 2.0/CLDC 1.1).

 

 

 

4) O WikiCrimes Mobile é compatível com o meu Celular?

O seu celular deve ser capaz de executar aplicativos JAVA (J2ME MIDP 2.0/CLDC 1.1 ou melhor configuração). Você pode verificar a configuração JAVA no próprio celular, nomanual do celular ou baixar um aplicativo gratuitamente que verificará se o seu celular é conpatível com o WikiCrimes Mobile. Testamos o WikiCrimes Mobile nos seguintes celulares: Nokia N80, Nokia N95, Nokia 3600, Nokia 2630, Nokia 6124 e Sony Ericsson W870i. Não é necessário que o seu celular possua GPS mas se possuir o WikiCrimes Mobile vai utiliza-lo para obter as coordenadas do local onde você se encontrará no momento da consulta a base de dados do WikiCrimes.org (www.wikicrimes.org).

 

 

 

5) Como funciona o serviço WikiCrimes Mobile?

Estas imagens são ilustrativas pois as fontes e tamanhos aqui apresentados,  podem variar dependendo do modelo do celular no qual a aplicação será instalada.

1º passo ) Uma vez instalado (como instalar?) WikiCrimes Mobile em seu celular, selecione a aplicação a partir do menu de aplicações em seu celular.

2º Passo ) Selecione o serviço  AQUI É SEGURO? .

3º Passo) Selecione o período de análise dos crimes.

4º Passo) Caso seu celular tenha o serviço de GPS ativo, seu celular identificará sua posição geográfica e essa etapa ocorrerá automaticamente não sendo necessário a exibição da tela a seguir. Caso a tela apareça em seu celular,  você deverá informar o local (rua/avenida... e número aproximado) para o qual deseja obter a informação.

5º Passo) WikiCrimes Mobile lhe informará para o período selecionado e a área identificada / informada os crimes ocorridos (roubos, furtos, homicídios e outros tipos de crime) em um raio de 500 m do ponto identificado / informado.
